# OrbitPMS

A modern Property Management System (PMS) built for managing hotel operations including room management, bookings, guest check-ins/check-outs, invoicing, and availability tracking.

## Project Overview

OrbitPMS is a full-stack hotel management application being developed to simulate real-world hospitality workflows.

The goal of this project is to understand:

* Property Management System (PMS) architecture
* Authentication and authorization
* Database relationships
* Booking and reservation workflows
* API integration
* Full-stack application development

## Planned Features

### Authentication

* User Registration
* User Login
* Logout

### Room Management

* Add, Edit, Delete Rooms
* Room Types
* Room Availability Status

### Booking System

* Create Reservations
* Assign Rooms
* Prevent Double Booking

### Check-In / Check-Out

* Guest Check-In
* Guest Check-Out
* Automatic Room Status Updates

### Invoice Generation

* Stay Summary
* Room Charges
* Total Billing

### Availability Calendar

* Room Availability Tracking
* Booking Visualization

## Tech Stack
