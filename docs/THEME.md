# OrbitPMS Theme System

## Design Philosophy

OrbitPMS is a professional Property Management System used by hotel receptionists, managers, and staff throughout the day.

The interface must prioritize:

* Fast operations
* Data readability
* Clear room status visibility
* Minimal clicks
* Enterprise appearance
* Responsive desktop-first workflow

Avoid:

* Excessive gradients
* Glassmorphism
* Consumer-style booking site layouts
* Large decorative animations
* Unnecessary shadows

# PMS UX Rules

Always show room status with color badges.

Never hide important actions inside dropdowns.

Booking creation must be possible within 2 clicks.

Check-In and Check-Out buttons must be visible in tables.

Search and filters must remain visible.

Critical hotel operations should never require scrolling to access.

Data tables take priority over decorative content.

OrbitPMS is an operational software product, not a marketing website.
